# jackhroberts.dev

Source for my personal site — hand-built with plain HTML, CSS, and JavaScript. No framework, no build step.

**Live:** [jackhroberts.dev](https://jackhroberts.dev)

## What's here

- `index.html` — the site: a year-by-year timeline of experience and projects, skills, and an interactive get-to-know-me section
- `Resume.html` + `pdfs/` — my resume and PDF write-ups/decks for selected projects (DataFest 2025, Furman Engaged, Maternal Health Risk Prediction, Spotify Success Prediction)
- `projects/` — standalone pages for those projects
- `css/`, `js/`, `images/` — styling, interactivity, and assets
- `CNAME` — deployed via GitHub Pages

## Running locally

No dependencies:

```bash
python3 -m http.server
# open http://localhost:8000
```
